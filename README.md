# C Practice

Learn basic syntax for c

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Sleepy4k/c-practice.git
~~~

Go to the project directory  

~~~bash  
  cd c-practice
~~~

Start the c file  

~~~bash  
  gcc <filename> -o <output name>
~~~

## Usage/Examples

~~~bash
  gcc hello.c -o hai
~~~  

~~~bash
  ./hai
~~~
